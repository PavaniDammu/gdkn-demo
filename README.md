# gdkn-demo
Nodejs +  Angular + Mysql

## Description

This is a full stack web application built with Nodejs, Angular and MySQL technologies.

Functionalities:
- Users can be added with details
- Search users
- Delete users
- Edit user info

## Project setup

### Node.js Server
```
cd demo-be
npm install
node server.js
```

### Angular Client
```
cd demo-fe
npm install
ng serve
```
